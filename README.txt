TechGuyTool – Android OTA Idiot Proof
====================================

TechGuyTool Android OTA Idiot Proof is a lightweight ADB-based utility
that disables Android system update (OTA) services.

No root required.
No permanent system modification.

This tool is intended for users who want full control over updates
and wish to prevent accidental or automatic OTA installation.

--------------------------------------------------
WHAT THIS TOOL DOES
--------------------------------------------------
• Clears pending OTA update state
• Disables OEM update services
• Prevents background or accidental updates
• Reboot finalizes the change

The operation is reversible by:
• Factory reset
• Full firmware flash
• Manually re-enabling packages via ADB

--------------------------------------------------
REQUIREMENTS
--------------------------------------------------
• Windows PC
• USB cable
• Android device
• USB Debugging enabled
• Device unlocked with RSA prompt accepted

--------------------------------------------------
HOW TO USE
--------------------------------------------------
1. Enable USB debugging on your phone:
   Settings → About phone → Tap Build number 7 times
   Settings → Developer options → Enable USB debugging

2. Connect the phone to the PC and unlock it

3. Run:
   TGT_OTA_Launcher.cmd

4. Select your device brand from the menu

5. Follow on-screen instructions

The device will reboot automatically when finished.
