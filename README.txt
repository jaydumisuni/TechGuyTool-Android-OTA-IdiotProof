TechGuyTool – Android OTA Idiot Proof
====================================

This tool disables Android OTA (system update) services using ADB.
No root required. No permanent system modification.

This is intended for users who:
- Do NOT want automatic updates
- Want full control over when (or if) updates install
- Understand the trade-offs of disabling updates

--------------------------------------------------
WHAT THIS TOOL DOES
--------------------------------------------------
- Clears pending OTA update state
- Disables OEM updater services
- Prevents accidental or automatic updates
- Reversible via factory reset or firmware flash

--------------------------------------------------
REQUIREMENTS
--------------------------------------------------
- Windows PC
- USB cable
- Android phone
- USB Debugging enabled
- User must ACCEPT the RSA prompt on the phone

--------------------------------------------------
HOW TO USE
--------------------------------------------------
1. Enable USB debugging on your phone
   Settings > About phone > Tap Build number 7 times
   Settings > Developer options > USB debugging

2. Connect phone to PC and unlock it

3. Run:
   TGT_OTA_Launcher.cmd

4. Select your device brand

5. Follow on-screen instructions

The device will reboot when finished.

--------------------------------------------------
NOTES
--------------------------------------------------
- On some devices, on-screen messages (toasts) may not appear.
  This is normal and depends on the system UI.

- Disabling updates means you will NOT receive:
  - Security patches
  - Feature updates
  - OEM fixes

You are responsible for this choice.
